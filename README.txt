

🏥 DataMart de Gestion Hospitalière

Ce projet BI a pour but de concevoir un DataMart complet pour la gestion hospitalière, depuis la modélisation des données jusqu’à la visualisation via Power BI. Il s'agit d'un travail réalisé dans un contexte académique pour simuler un environnement décisionnel réel.

📌 Objectif du projet

Offrir au directeur médical un système d’aide à la décision lui permettant de suivre les indicateurs clés de performance hospitalière : taux d’occupation des lits, nombre de patients, durée moyenne de séjour, performance des services médicaux, etc.

---

🧱 Étapes du projet

1. Choix du thème : Gestion hospitalière
2. Définition de la table de faits et des dimensions
3. Identification des mesures importantes
4. Construction du dictionnaire de données
5. Modélisation du schéma en étoile
6. ETL avec SSIS : extraction des données depuis le système OLTP vers le Data Mart
7. Création du cube OLAP avec SSAS
8. Élaboration d’une matrice d’alignement stratégique
9. Conception d’un tableau de bord interactif dans Power BI

---

🧰 Technologies utilisées


- SQL Server Management Studio(base de donne source et destination)
- visuel studio  SSIS (SQL Server Integration Services)
- visuel studio  SSAS (SQL Server Analysis Services)
- Power BI
